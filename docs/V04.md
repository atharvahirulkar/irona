# Cadbury v0.4

## Added

| Feature | Command / config |
|---------|----------------|
| **Generation eval** | `cadbury eval --generation` |
| **WhatsApp draft** | `whatsapp.draft` in `enabled_tools`, `cadbury whatsapp +1… "msg"` |
| **Voice hints** | `voice_enabled: true`, `/listen`, `/voice on` |
| **Auto-approve for eval** | `CADBURY_AUTO_APPROVE=1` (set by generation eval) |

## WhatsApp honesty

v0.4 does **not** auto-send WhatsApp messages. It opens a draft; you tap Send. See [WHATSAPP.md](WHATSAPP.md).

## Suggested config snippet

```yaml
enabled_tools:
  - calendar.read
  - web.search
  - whatsapp.draft

voice_enabled: true   # after: pip install faster-whisper sounddevice soundfile
```

## Run

```bash
pip install -e .
cadbury eval
cadbury eval --generation --limit 3   # quick LLM smoke test
cadbury whatsapp +1XXXXXXXXXX "test message"
cadbury start   # /listen, /voice on
```
